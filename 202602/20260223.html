[테크] Transformers.js v4 프리뷰, NPM 설치 지원 시작

2026년 02월 9일

허깅페이스의 Transformers.js v4 프리뷰가 NPM에 등록됐어요. Transformers.js는 허깅페이스의 Transformers 라이브러리를 웹 브라우저에서 실행할 수 있도록 만든 자바스크립트 라이브러리인데요. 기존에는 GitHub에서 소스코드를 직접 다운받아 설치해야 했지만, 이제 NPM 명령어 한 줄로 설치할 수 있게 됐습니다.

가장 큰 변화는 C++로 완전히 재작성된 새로운 WebGPU 런타임이에요. ONNX 런타임 팀과 협력해 개발된 이 런타임은 약 200개 모델 아키텍처를 지원하고요. 동일한 코드를 웹 브라우저뿐 아니라 Node, Bun, Deno 등 다양한 자바스크립트 환경에서 사용할 수 있어요. 최신 AI 모델을 브라우저 로컬 환경에서 실행할 수 있다는 것을 입증한 단계이고, 현재는 성능 최적화에 집중하고 있다고 해요.
